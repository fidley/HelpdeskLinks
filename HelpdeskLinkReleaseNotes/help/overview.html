Helpdesk Links Plugin - Overview

The Helpdesk Links plugin provides automatic hyperlinking for helpdesk-related references in your code, such as Jira tickets, GitHub issues, SAP Notes, Ivanti requests, and more. It allows you to configure custom patterns and link templates, and to choose how links are opened (external browser, internal browser, or Eclipse default).

Main Features

  • Automatic detection and hyperlinking of helpdesk references
  • Customizable regex patterns and link templates
  • Support for Jira, GitHub, SAP Notes, Azure DevOps, Ivanti, Email, and generic URLs
  • Preference page for managing patterns and link templates
  • Option to choose browser for opening links
  • Import/export of pattern configurations
