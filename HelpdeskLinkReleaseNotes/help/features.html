Features

  • Automatic hyperlinking for Jira, GitHub, SAP Notes, Azure DevOps, Ivanti, Email, and generic URLs
  • Customizable regex patterns and link templates
  • Import/export of pattern configurations
  • Preference page for browser selection (External, Internal, Eclipse Default)
  • Scope patterns to projects or working sets
  • Easy-to-use UI for managing patterns and links
