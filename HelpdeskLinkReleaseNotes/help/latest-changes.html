Version 0.0.1 - initial version

After installing the plugin, you can access the preferences via Window → Preferences → Helpdesk Links. Here you can manage your patterns and link templates.

  • Add, Edit, Remove, Copy allows to manage your patterns and link templates.
  • Import/Export: Import and export of hyperlink pattern configurations, so you can share it and make backups.

In the Browser section you can choose how links are opened:

  • External Browser: Opens links in your system's default web browser.
  • Internal Browser: Opens links in the Eclipse internal browser.
  • Eclipse Default: Uses the default behavior of Eclipse for opening links.

To add a new pattern or link template, click the Add button. This opens a dialog where you can enter the regex pattern and the link template.

There are predefined search and link patterns, but you can also use totally custom ones. Please check those regex examples.

You can also decide if the entry should be linked with selected projects, working sets or for all projects.
